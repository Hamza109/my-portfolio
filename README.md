# Gamer Portfolio (Next.js + Tailwind + shadcn-style components)

## Run

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

## Customize with your resume

Update portfolio content in:
- `/Users/hamzahamal/Desktop/portfolio/lib/portfolio-data.ts`

Replace placeholders for:
- email
- location
- projects
- experience timeline
- skills

Your PDF is available at:
- `/Users/hamzahamal/Desktop/portfolio/resume.pdf`

The hero "Download Resume" button already points to `/resume.pdf`.
